CacheAndNetwork

CacheAndNetwork()
Content copied to clipboard

Try the cache, then also try the network.

If the data is in the cache, it is emitted, if not, no exception is thrown at that point. Then the network call is made, and if successful, the value is emitted, if not, either an ApolloCompositeException (both cache miss and network failed) or an ApolloException (only network failed) is thrown.
